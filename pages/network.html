Web Server : A web server is a software and hardware that uses HTTP and other protocols to respond to client requests.
Server : A server is a computer program or device that provides a service to another computer program and its user also known as the client.
DNS Server : DNS servers translate domain names into IP addresses.
IP : An internet protocol address is the unique identifying number assigned to every device connected to the internet.
Domain : A domain name is a text string that maps to an IP address used to access websites.
